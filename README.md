# speech-to-text-to-speech-AI
Assistente de voz em Python no Google Colab que grava áudio no navegador, transcreve com Whisper, gera respostas com modelos gratuitos da Hugging Face e converte texto em fala em português. Projeto simples e acessível para iniciantes, integrando reconhecimento de fala, NLP e síntese de voz em um fluxo completo.
